Developers API

It exposes services to facilitate integration with third-party applications.

POST​/rowcreate
Create a new record in a table/entity. This operacion is a shortcut to use a commit request.
POST​/rowupdate
Update a record in a table/entity. This operacion is a shortcut to use a commit request.
POST​/rowdelete
Delete a record in a table/entity. This operacion is a shortcut to use a commit request.
